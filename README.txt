Este projeto foi desenvolvido como parte do processo seletivo para a vaga de Desenvolvedor React-Native, referente a empresa Rubcube.

Antes de executar o projeto, execute npm install (ou yarn install), para baixar as dependências.

O projeto foi primariamente desenvolvido e testado em um emulador android (Pixel 3).

O aplicativo possibilita a pessoa favoritar usuários do Github, conseguindo consultar seus repositórios e detalhes sobre cada um.
